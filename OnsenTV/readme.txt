================================================================================
　温泉旅館TV
（フリーソフト）
================================================================================

■概要
　１回起動すると１０分でブラウザが強制終了するアプリケーションです。

　　１．起動するブラウザは以下から選べます。
　　　　・Internet Explorer
　　　　・Mozilla firefox
　　　　・Google Chrome
　　　　・Opera
　　２．強制終了までの時間を選べます。
　　３．強制終了後しばらくはこのアプリケーションを起動できません。

------------------------------------------------------------
■！！！注意！！！
　一定時間後にブラウザが　＜※強制終了＞　しますので、
　終了して困るような作業をこのアプリケーションで
　行わないでください。
------------------------------------------------------------

■動作環境
　Windows Vista, Windows XP
　(Microsoft .NET Framework Version 2.0 が必要です)

■インストール方法
　温泉旅館TV.exe、温泉旅館TV設定.exe、Interop.SHDocVw.dll を
　適当なフォルダに保存してください。

■アンインストール方法
　上記プログラムを削除して(ごみ箱へ入れて)ください。

　必要であれば以下の場所に保存された設定ファイルも削除してください。

　Windows Vista:
　　C:\Users\<UserName>\AppData\Local\OnsenTV

　Windows XP:
　　C:\Documents and Settings\<UserName>\Local Settings\Application Data\OnsenTV

■使い方
　温泉旅館TV.exe を起動します。

　初回起動時は設定ダイアログが表示されます。

　設定内容：
　　・起動するブラウザ
　　・制限時間
　　・待ち時間
　　・音を鳴らすかどうか

　OK ボタンでブラウザが起動します。
　制限時間後に強制終了します。
　待ち時間が経過するまで再起動できません。

　温泉旅館TV設定.exe を起動すると設定した内容を変更できます。
